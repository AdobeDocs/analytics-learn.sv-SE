# Segmentering på antal distinkta Dimensioner {#segmentation-on-distinct-dimension-counts}

De nya [!UICONTROL Distinct Count] operatorerna i [!UICONTROL Segment builder] gör att du kan segmentera baserat på antalet unika objekt i ett [!UICONTROL dimension]. Filtrera efter besök där mer än fem sidor visats, eller träffar där mer än fem produkter köptes, dvs. i en enda beställning.
