# Kör om ett datafeedjobb

Användargränssnittet för dataflöden i Adobe Analytics gör dataflödeshanteringen mycket enklare. Om du har tillgång till att hantera dataflöden kan du köra ett datafeedjobb på nytt, på egen hand, i användargränssnittet för dataflöden, utan att kundsupportteamet behöver skicka om dataflödesfilerna.
